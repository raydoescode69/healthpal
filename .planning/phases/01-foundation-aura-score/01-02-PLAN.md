1. Create `lib/auraCalculator.ts` with a `calculateAuraScore` function:

   **Input:** `{ calories, protein_g, carbs_g, fat_g, waterGlasses, steps, sleepHours }`
   (sleepHours defaults to 7 if not tracked -- sleep tracking is not yet built, use a reasonable default)

   **Scoring algorithm (each sub-score 0-100, overall = weighted average):**
   - **Sleep (25% weight):** 8h = 100, scale linearly. <4h = 0, >10h caps at 85 (oversleep penalty).
   - **Nutrition (30% weight):** Based on calorie target adherence. If user ate 80-120% of 2000 cal target = 100. Farther from target = lower. Also factor protein ratio (>30% of cals from protein = bonus 10 points, capped at 100).
   - **Movement (25% weight):** steps / 10000 * 100, capped at 100.
   - **Hydration (20% weight):** waterGlasses / 8 * 100, capped at 100.

   **Output:** `{ score, sleepScore, nutritionScore, movementScore, hydrationScore, label, description }`

   **Label mapping:**
   - 75-100: "THRIVING" / "You slept well, ate clean, and moved. Keep this up."
   - 50-74: "RESTING" / "Decent day. A few tweaks and you'll be glowing."
   - 25-49: "PUSHING" / "Room to improve. Focus on hydration and movement."
   - 0-24: "STRUGGLING" / "Rough day. Tomorrow's a fresh start."

   **Color mapping (export as helper):**
   - 75-100: "#bef135" (lime)
   - 50-74: "#38bdf8" (blue)
   - 25-49: "#fb923c" (orange)
   - 0-24: "#ef4444" (red)

   Export: `calculateAuraScore`, `getAuraColor`, `getAuraLabel`

2. Create `components/AuraOrb.tsx`:
   An animated SVG orb component using Reanimated v4 + react-native-svg.

   **Props:** `score: number` (0-100), `color: string` (from getAuraColor), `size?: number` (default 240)

   **Visual structure (matching HTML mockup):**
   - Outer SVG container sized to `size` x `size`
   - 3-4 concentric circles with decreasing opacity to create glow effect:
     - Outermost: radius = size/2, fill = color at 0.08 opacity, animated scale pulse (withRepeat, 1.0 to 1.05, 3000ms)
     - Middle: radius = size/2 * 0.75, fill = color at 0.15 opacity, animated scale pulse (offset timing)
     - Inner: radius = size/2 * 0.5, fill = color at 0.25 opacity
     - Core: radius = size/2 * 0.35, fill = color at 0.4 opacity
   - Use `useAnimatedProps` for SVG animations (runs on UI thread, per STACK.md guidance)
   - Use `Animated.createAnimatedComponent(Circle)` for animated circles

   **Score display (overlaid on orb):**
   - Large score number (e.g., "84") centered, white, fontSize 48, fontWeight bold
   - "/ 100" suffix, smaller, rgba(255,255,255,0.5)
   - These are positioned absolutely over the SVG using a View overlay

   Do NOT render the label or description inside the orb component -- the parent screen handles those.
