**2. Create `app/(main)/bet-detail.tsx`** — The complete bet detail screen.

Get `id` from route params: `const { id } = useLocalSearchParams<{ id: string }>()`.
Get userId from `useAuthStore(s => s.session?.user?.id)`.
Get theme from `useThemeStore` + `THEMES`.
Get bets state from `useBetsStore`.

**Lifecycle (CRITICAL for realtime — follow PITFALLS.md):**
- Use `useFocusEffect` from `expo-router` (which re-exports from React Navigation).
- On focus: call `loadChallenge(id)`, then `subscribeToActive(id)`.
- On blur (cleanup): call `unsubscribeFromActive()`.
- This prevents the Supabase Realtime connection leak pitfall.
- Also handle app state changes: import `AppState` from `react-native`. On 'active' state, re-subscribe. On 'background', unsubscribe. Use `useRef` for the AppState subscription.

**Countdown Timer:**
- Create a `useCountdown(expiresAt: string)` custom hook inline or at top of file.
- Uses `useState` for `{ hours, minutes, seconds }` and `useEffect` with `setInterval(1000)`.
- Calculate remaining: `new Date(expiresAt).getTime() - Date.now()`.
- If remaining <= 0, return `{ hours: 0, minutes: 0, seconds: 0, isExpired: true }`.
- Format each with `.toString().padStart(2, '0')`.
- Clean up interval on unmount.
